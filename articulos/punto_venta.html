Punto de Venta

Se creo un punto de venta para la refaccionaria El Gran Pez, el cual tiene como objetivo llevar un control de los productos que se venden y de los productos que se compran. El sistema tiene la capacidad de generar reportes de ventas y compras, así como de mostrar el inventario actual de la tienda.

El sistema fue desarrollado utilizando React para el frontend, por otro lado se uso Python y PHP para el backend. Esto debido a que ciertas partes del sistema requieren de un procesamiento de datos que es mejor realizar en Python, como el análisis de ventas y compras.
